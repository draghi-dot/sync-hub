# Access Codes Reference

This document lists all access codes used in the system.

## Company Access Codes

These codes are required when setting up or changing your company in your profile:

- **Adobe**: `AdobeCode`
- **Google**: `GoogleCode`
- **Electronic Arts**: `EACode`

## Department Access Codes

These codes are **standardized across all teams**. For example, "CyberSecurity" has the same code whether it's in Photoshop, Illustrator, InDesign, Google Drive, etc.

### Standardized Department Codes

All departments with the **same name** use the **same access code** across all teams:

- **Software Development**: `DEV-2847`
- **Marketing**: `MKT-6193`
- **CyberSecurity**: `SEC-9251`
- **Finance**: `FIN-4738`
- **Design**: `DSG-8462`

### Examples

- If you want to join "CyberSecurity" in **Photoshop**, use code: `SEC-9251`
- If you want to join "CyberSecurity" in **Illustrator**, use code: `SEC-9251`
- If you want to join "CyberSecurity" in **Google Drive**, use code: `SEC-9251`
- If you want to join "CyberSecurity" in **EA Sports FC**, use code: `SEC-9251`

**The same code works for all teams!**

### How to Use

1. Go to your Profile â†’ Edit Profile
2. Select your Company and enter the company code (e.g., `AdobeCode`)
3. Select a Department from the dropdown
4. Enter the department access code (e.g., `SEC-9251` for CyberSecurity)
5. Click "Add" to add the department
6. You can add multiple departments using their respective codes

## Notes

- Once you're assigned to a department, you don't need the code to access it again
- You can be assigned to multiple departments
- Access codes are case-sensitive
- If you forget a code, refer back to this document
