# Access Codes Reference - Updated

This document lists all access codes used in the system.

## Company Access Codes

These codes are required when setting up or changing your company in your profile:

- **Adobe**: `AdobeCode`
- **Google**: `GoogleCode`
- **Electronic Arts**: `EACode`

## Department Access Codes

Access codes are **company-specific**. Each company has its own set of codes for departments.

### Adobe Company Codes (AD-*)

All Adobe departments (Photoshop, Illustrator, InDesign) use these codes:

- **Software Development**: `AD-DEV-1000`
- **Marketing**: `AD-MAR-1000`
- **CyberSecurity**: `AD-SEC-1000`
- **Finance**: `AD-FIN-1000`
- **Design**: `AD-DES-1000`

**Examples:**
- CyberSecurity in Photoshop: `AD-SEC-1000`
- CyberSecurity in Illustrator: `AD-SEC-1000`
- CyberSecurity in InDesign: `AD-SEC-1000`

### Google Company Codes (GO-*)

All Google departments (Google Drive, Google Slides, Google Gemini, YouTube) use these codes:

- **Software Development**: `GO-DEV-1000`
- **Marketing**: `GO-MAR-1000`
- **CyberSecurity**: `GO-SEC-1000`
- **Finance**: `GO-FIN-1000`
- **Design**: `GO-DES-1000`

**Examples:**
- CyberSecurity in Google Drive: `GO-SEC-1000`
- CyberSecurity in YouTube: `GO-SEC-1000`

### Electronic Arts Company Codes (EA-*)

All Electronic Arts departments (EA Sports FC, Apex Legends, The Sims) use these codes:

- **Software Development**: `EA-DEV-1000`
- **Marketing**: `EA-MAR-1000`
- **CyberSecurity**: `EA-SEC-1000`
- **Finance**: `EA-FIN-1000`
- **Design**: `EA-DES-1000`

**Examples:**
- CyberSecurity in EA Sports FC: `EA-SEC-1000`
- CyberSecurity in Apex Legends: `EA-SEC-1000`

## How to Use

1. Go to your Profile â†’ Edit Profile
2. Select your Company and enter the company code (e.g., `AdobeCode`)
3. Select a Department from the dropdown
4. Enter the department access code based on your company:
   - **Adobe**: `AD-[DEPT]-1000` (e.g., `AD-SEC-1000` for CyberSecurity)
   - **Google**: `GO-[DEPT]-1000` (e.g., `GO-SEC-1000` for CyberSecurity)
   - **Electronic Arts**: `EA-[DEPT]-1000` (e.g., `EA-SEC-1000` for CyberSecurity)
5. Click "Add" to add the department
6. You can add multiple departments using their respective codes

## Notes

- Codes are **case-sensitive**
- Once assigned to a department, you don't need the code to access it again
- You can be assigned to multiple departments
- Codes are the same for a department name within the same company
